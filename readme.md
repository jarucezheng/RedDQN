# 红方Agent v1.0

### 设计思路

这个Agent的目标是，当发现了敌方的飞行单位时，由Agent决定派出哪架飞机去打。

##### 输入空间：1.我方所有战斗机距离此目标的距离；2.我方所有现存战斗机现有对空导弹数；3.我方所有现存战斗机燃油量；4.特别的，利用0-1mask来表示战斗机是否存活

##### 输出空间：每架战斗机出战的Q值，选出最大的index
